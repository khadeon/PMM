1.- DIFERENCIAS ENTRE TOSTADAS y DIALOGOS
Vamos a tener informado al usuario de dónde estamos en cada momento. Para ello se usan dos tipos de mensajes: TOSTADAS y DIALOGOS

 protected void showToast(CharSequence text) {
        Context context = getApplicationContext();
        int duration = Toast.LENGTH_SHORT;
        Toast toast = Toast.makeText(context, text, duration);
        toast.show();
    }

protected void showAlert( CharSequence text) {
         AlertDialog.Builder alert = new AlertDialog.Builder(this);
        alert.setMessage(text);
        alert.setPositiveButton(android.R.string.ok, null);
        alert.show();
    }

2.- Indroduce Un recurso Musical en nuestros primeros ejercicios
HolaMundos..

Crea un directorio (raw) dentro de res (recuerda que los nombres siempre en minusculas, los recursos NO pueden contener  mayúsculas). Copia el fichero desigual.mp3 en el directorio creado. Dentro del método onCreate crea una variable del tipo MediaPlayer de esta manera: 

MediaPlayer miMusica= MediaPlayer.create(getApplicationContext(),R.raw.desigual);
miMusica.start();

   Toast.makeText(this,"onResume", Toast.LENGTH_SHORT).show();

 }
